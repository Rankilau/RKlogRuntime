简介
RKlogRuntime是一个免费且开源的无痕埋点方法，实现按钮点击事件无痕埋点，viewcontroller的pv无痕埋点。目的是给开发者提供一个无痕埋点案。

常见问题

RKlogRuntime组件收费吗？
RKlogRuntime是完全开源并且免费给开发者使用。

接入RKlogRuntime有什么好处？
组件可以实现无痕埋点，对代码侵蚀性减到最小；实现动态下发埋点逻辑，防止遗漏重要埋点，影响业务统计。
